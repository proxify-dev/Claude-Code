# Skills Catalog

Recommended skills for Proxify engineers using Claude Code.

## Core Skills

| Skill | Description | Install Command |
|-------|-------------|-----------------|
| [skill-architecture](skill-architecture/SKILL.md) | How to design skills that reach agents at the right time. Covers context efficiency, distribution layers, and framing. | `cp -r skills/skill-architecture ~/.claude/skills/` |
| [agent-development](agent-development/SKILL.md) | Agent structure, system prompts, triggering conditions, and MCP server configuration for Claude Code plugins. | `cp -r skills/agent-development ~/.claude/skills/` |
