## How to use

- Pick the style that matches your reader's context — someone scanning for a quick answer needs a different format than someone learning the concept for the first time
- Copy the file, replace the Permutation Frameworks content with your topic, keep the structure
- Each style showcases specific Mintlify components — the component coverage table below shows which style to look at for any given component

| Component | Styles |
|-----------|--------|
| `<Tip>` | Quick Win, Cookbook, Cheatsheet, Pair Programming |
| `<Note>` | Narrative, Reference, Socratic, Mentor's Margin Notes |
| `<Warning>` | Quick Win, Cookbook, Reference, Anti-Pattern First |
| `<Check>` | Cookbook, Anti-Pattern First, Checklist Only |
| `<CardGroup>` / `<Card>` | This page, War Story |
| `<Steps>` / `<Step>` | Pair Programming |
| `<Tabs>` / `<Tab>` | Before / After |
| `<AccordionGroup>` / `<Accordion>` | Progressive Disclosure |
| `<CodeGroup>` | Cheatsheet |
| `<Frame>` | Annotated Example |
| `<Expandable>` | Mentor's Margin Notes |
| Mermaid diagrams | Decision Tree |
